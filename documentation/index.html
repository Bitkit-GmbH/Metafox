Folder structure

                                HTML/
                                |-- css/
                                |   |-- .-> All CSS files
                                |-- fonts/
                                |   |-- .-> All fonts
                                |-- images/
                                |   |-- .-> All images
                                |-- js/
                                |   |-- .-> All javascript files
                                |-- php/
                                |   |-- .-> All PHP files
                                |-- scss/
                                |   |-- .-> All SCSS files
                                |-- vendor/
                                |   |-- .-> All third party files
                                |-- videos/
                                |   |-- .-> All videos
                                `-- *.html

CSS

The files are described in the table below:

File Description
master.css This is main CSS file for the theme, which includes the layout and styling for all aspects of the theme.
color/*.css These are CSS files for the theme color. The theme contains 6 predefined colors, which can be linked in the <head> area. If you want to use your own color, you can create a new CSS file and name it after your color. To do this, simply copy the content of an existing color file, change the hex color values and link them in the <head> area.
SCSS

The files are described in the table below:

File Description
*.scss If you would like to write your CSS code in Sass, you will find the files in the SCSS folder. Each section is saved as a single file.
Javascript

The files are described in the table below:

File Description
main.js This is the main Javascript file. It contains all the custom code that is used for features.
contact.js This is the Javascript file for the contact form and the Google Recaptcha verification.
PHP

The files are described in the table below:

File Description
contact.php This is the PHP file which is necessary for sending emails. Make sure you define the recipient address, Google Site Key and Google Secret Key before using the contact form. You can create the site key and the secret key via the following website: https://www.google.com/recaptcha. Important: Be sure to select Recaptcha version 2.
